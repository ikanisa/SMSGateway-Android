# SMS Gateway User Guide

Welcome to SMS Gateway! This guide helps you get started with using the app.

---

## Getting Started

### Installation
1. Open the download link from your email
2. Tap "Download" to get the APK
3. If prompted, allow "Install from unknown sources"
4. Install the app

### First-Time Setup
1. Open SMS Gateway app
2. Grant SMS permission when prompted
3. The app will show "Listening" status

---

## Using the App

### Dashboard
The main screen shows:
- **Status** - Green dot = Active, Red = Stopped
- **Processed** - Number of SMS forwarded
- **Errors** - Any failed forwarding attempts
- **Logs** - Recent activity

### Permissions
The app needs:
- **SMS Permission** - To read incoming messages
- These are ONLY used to forward to your backend

---

## FAQ

**Q: Is my data secure?**
A: Yes. SMS are encrypted in transit and the app uses secure storage.

**Q: Does this work without internet?**
A: SMS are queued when offline and sent when connection is restored.

**Q: How do I stop forwarding?**
A: Revoke SMS permission in device Settings, or uninstall the app.
